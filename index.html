Web sitelerinin temelini HTML oluşturur. Tüm web sitelerinin (microsoft.com, facebook.com, vb.) arayüzü hep HTML ile oluşturulmuştur. HTML kursumuzda daha çok HTML in temel yapısını oluşturan etiketler üzerinde duracağız. Fazla detaylara girilmeden, örnekler ile kolay ve pratik olarak HTML i hep beraber öğreneceğiz. Bu kursun ana amacı kendi web sitelerinizi oluşturacak altyapıyı sizlere sunmaktır. Bu kursu bitirdikten sonra; HTML etiketlerini tanıyor olacaksınız. Bir web sayfasının nasıl oluşturulduğunu öğrenmiş olacaksınız. Statik ve dinamik web sayfaları arasındaki farkı anlamış olacaksınız. HTML kursu, HTML Temel ve HTML + CSS şeklinde iki seri halinde olacaktır. Bu kursumuz daha çok teorik ağırlıklı ve sadece HTML etiketleri üzerine olmasına rağmen, diğer kursumuz ise HTML + CSS etiketleri üzerine detaylı ve uygulama ağırlıklı olacaktır. Bu iki kursu bitirdikten sonra profesyonel web sitelerinin nasıl tasarlandığını öğrenmiş olacaksınız ve kendi web sitenizi tasarlamaya başlayabileceksiniz. Hedef kitle kimdir?
